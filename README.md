# Smart City Data Analyzer

This project is an AI-powered Smart City Data Analyzer built using Python and Streamlit.

## Features
- Upload urban datasets
- Traffic analysis
- Pollution visualization
- Correlation analysis
- Key insights generation

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
